📖 How to Set Up Google Sheets
1

Create a Google Spreadsheet

Go to sheets.google.com and create a new spreadsheet. Name it "Restaurant Operations".

2

Create 5 Sheet Tabs

Create tabs named exactly: Inventory, StockReceived, StockUsage, Expenses, Income.

3

Add Column Headers

In each sheet, add the headers in Row 1 exactly as specified. The app will read/write from Row 2 onwards.

4

Open Apps Script Editor

In your spreadsheet, go to Extensions → Apps Script. Delete any existing code.

5

Paste the Apps Script Code

Copy the complete code from the section below and paste it into the Apps Script editor.

6

Deploy as Web App

Click Deploy → New deployment. Set type to Web app. Set "Execute as" to Me and "Who has access" to Anyone. Click Deploy and copy the URL.

7

Paste URL Above

Paste the Web App URL in the configuration field above and click Save URL.
